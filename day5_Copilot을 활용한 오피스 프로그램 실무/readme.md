# day5 Copilot을 활용한 오피스 프로그램 실무

## 워드 업무 효율화
1.1 데이터 분석 프롬프트

```
첨부한 데이터는 우리 회사의 6월달 소비 지출 내역입니다. 먼저 첨부한 데이터를 기반으로 우리 회사의 소비 패턴을 분석해 주세요. 
그 다음 분석한 내용을 기반으로 지출을 줄일 수 있는 아이디어 세 개를 작성해주세요. 소비 패턴을 분석하기 전까진 아이디어를 도출하지 마세요. 
```

1.2 비용 절감 시뮬레이션 프롬프트

```
사무실 내 커피 머신을 도입한다면 커피 지출을 경감할 수 있을까요?
우리 회사의 커피 지출이 월 80만원이라고 가정했을 때 100만원대 커피 머신을 도입하면 얼마의 지출을 경감할 수 있는지 시뮬레이션해서 알려주세요.
참고로 커피 머신 유지 비용은 월 10만원 입니다.
시뮬레이션을 하는 과정에서 추가로 고려해야하는 사항이 있다면 답변하지 말고
먼저 추가 질문 후, 저의 답변을 듣고 결과를 말해주세요.
```

```
아래의 경우 월 절감액을 시뮬레이션 하시오.
- 커피 머신 가격: 1,000,000원
- 월 유지비: 100,000원
- 외부 커피 평균 단가: 5,000원
- 내부 제조 커피 평균 단가: 800원
- 월 소비 잔수: 160잔
- 머신 도입 후 외부 구매 대체율: 90%
- 감가상각 기간: 24개월
```

1.3 시장 조사 프롬프트

```
시뮬레이션에 기반해 사내 커피 지출을 줄이기 위한 100만원대 커피 머신을 도입하고자 합니다. 
네이버 블로그에서 아래 요구사항을 충족하는 글을 검색 후 추천해주세요.
요구사항 1.
'쿠팡' 이라는 단어가 제목에 포함된 블로그 글 6개를 검색하세요. 
요구사항 2. 
각 블로그 글에서 추천하는 커피 머신 중 100만원이하의 제품들만 필터링해주세요. 
요구사항 3. 
각 제품의 이름과 가격, 블로그 출처를 표로 정리해주세요.
각 요구사항을 단계별로 수행해주세요. 요구사항을 충족하지 못했다면 다음 요구사항으로 넘어가지 마세요.
```

1.5 보고서 작성 프롬프트

```
당신은 조달팀 과장 입니다. 앞서 분석한 소비 패턴과 지출을 줄일 수 있는 아이디어를 기반으로 기획서를 작성해주세요.
요구사항 1.
기획서는 서론-본론-결론으로 구성해주세요.
요구사항 2.
6월달 총 지출 금액을 계산해주세요. 그 중 아메리카노(커피) 지출이 얼마나 큰 비중을 차지하는지 서론에 언급해, 커피 머신 도입의 필요성을 소개하는 서론을 300자 이내로 작성하세요.
요구사항 3.
본론은 커피 머신 제품 비교, 선택한 모델 소개 및 선택 근거, 커피 머신 구매 기대 효과로 구성해주세요.
요구사항 4.
검색한 100만원대 제품들의 가격과 특징을 비교해주고, 그 중 드롱기 전자동 커피 메이커를 선택한 이유를 커피 지출 빈도와 연결시켜 설득력 있게 300자 이내로 작성해주세요.
요구사항 5.
해당 제품을 구매했을 때 기대효과(금전적 효과와 복지 효과 등) 3개를 300자 이내로 작성해주세요. 
지출 경감 시뮬레이션했던 내용을 드롱기 전자동 커피 메이커 가격을 넣어 다시 시뮬레이션을 하고, 그 결과를 금전전 효과 파트에 작성해주세요.
요구사항 6. 
결론 파트는 앞선 내용을 기반으로 자유롭게 작성해주세요.
요구사항 7.
기획서는 사무적이고 진중한 톤을 유지해 주세요.
요구사항 8.
그 외에 다른 말은 하지 마세요.
```

## PPT 업무 효율화

2.1 발표 슬라이드 목차 구성 프롬프트

```
당신은 조달팀 과장 입니다. 우리 회사의 6월달 지출 내역 데이터를 기반으로 커피 머신 구매 건의 발표 슬라이드 10장을 한국어로 구성해주세요. 

요구사항 1. 
먼저 발표 제목과 발표자(김은행) 정보가 적힌 슬라이드로 발표를 시작하세요.

요구사항 2. 
우리 회사의 6월달 지출 중 커피(아메리카노) 지출이 몇 퍼센트나 차지하는지 계산해주세요. 그리고 커피 지출이 얼마나 큰 비중을 차지하는지 퀴즈로 발표를 시작해주세요. 커피 지출 비율을 계산하기 전까지 발표 슬라이드를 구성하지 마세요.

요구사항 3.
커피 머신을 구매해 지출을 줄일 수 있다는 제안을 하세요.

요구사항 4. 
커피 머신 구매 시 우리 회사에 어떤 기대효과를 가져올 수 있는지 3가지 객관적인 수치를 제시하세요.

요구사항 5. 
발표 마지막엔 경청해 줘 감사하다는 메시지를 남겨주세요. 

요구사항 6.
발표 슬라이드 내 종결 어미는 ‘-함’, ‘-임’ 으로 끝내주세요.

요구사항 7. 
발표 슬라이드 구성 외에는 다른 말은 하지 마세요

```

2.2 발표 스크립트 초안 작성 프롬프트

```
만든 발표 슬라이드를 기반으로 발표 대본을 작성해주세요.
아이디어 소개 및 기대효과를 설명하는 슬라이드는 최소한 500자 이상으로 대본을 작성하세요.
```
